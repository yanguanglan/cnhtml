标签，属性和元素

虽然html基础是纯文本，我们还是需要增加一些东西让它变得耀眼。

HTML文件的基础架构包括：标签及其中的内容。 将文件改成一下这样： This is my first web page 再次保存。回到网页浏览器，刷新页面。 网页表面没有任何改变，但是HTML的目的是赋予意义，不是呈现。这个例子已经定义了网页一些基本元素。 最上面的第一行，,是一个文件类型的申明。它同时告诉浏览器你偏好哪种HTML(这里是html5)。坚持申明很重要。如果你不这么做，浏览器会质疑你是否真知道自己在干什么，它也会不按常理出牌。 回到主题，是开始标签把所有东西赶在一起，然后告诉浏览器，在自己和闭合标签之间的内容是一个HTML文件。在和之间的是将在浏览窗口出现的文件的主要内容。

标签

HTML文件的基本结构就是用标签标记内容。

将文件改成如下：


<!DOCTYPE html>
<html>
<body>
    This is my first web page
</body>
</html>

再次保存。回到浏览器，重新页面

网页表面没有任何改变，但是HTML的目的是语意化定义内容结构，不是呈现。这个例子已经定义了网页一些基本元素。

最上面的第一行， <!DOCTYPE html>, 是一个 文档类型声明 它同时告诉浏览器你偏好哪种HTML(这里是html5)。 坚持申明很重要。如果你不这么做，浏览器会质疑你是否真知道自己在干什么，它也会不按常理出牌。

回到主题， <html> 是 起始标签 包含内容直到 </html> 闭合标签 之间的内容就是HTML文档内容。 而 <body> 到 </body> 是文档的主要内容，这部分内容会在浏览器里呈现出来。

闭合标签

</body> 和 </html> 闭合标签与之对应着起始标签。

不是所有的标签都这么闭合 (<html></html>) 有些没有内容的标签将会自行闭合，拿换行符line-break举例，像这样： <br> —换行符没有内容做靠山，所以只能自己悲催一个人呆着。我们一会再看几个例子。你要记住所有有内容的标签都必须闭合。格式如下：开始标签——内容——闭合标签。 严格来说，这不是必需要求，但是这是个惯例，当我们用这个教程时，它是个让我们更干净、容易理解代码的练习。

属性

标签也有属性，这让它们赋有更多的信息。属性在开始标签中出现，他们的值由双引号包围。举例如下<tag attribute="value">Margarine</tag>。过后再详细介绍。

再次提醒。双引号不是必要的。但是网站为稳定和干净，这是个良好练习习惯。照着做！

元素

标签主要标记一个元素的开始和结束。元素是编造网页的组成部分。你可以说，所有在<body> 和 </body> 标签间（包括）都是主体元素。再举例， 鉴于<title>和</title> 是 标签, <title>Rumple Stiltskin</title>是一个标题元素.
