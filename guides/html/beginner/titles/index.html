页面标题

所有html网页都应该有个标题。

在网页增加标题，代码更改如下：


<!DOCTYPE html>
<html>
<head>
    <title>My first web page</title>
</head>
<body>
    This is my first web page
</body>
</html>

我们增加了新的元素，从head和title标签开始（注意他们如何闭合）

元素head（ <head>开始 </head>闭合中间部分内容 ）在body元素前出现（ <body>开始 </body>闭合中间部分内容 ）,包含了网页的相关信息。head元素里的信息不会在浏览器中出现。

我们一会能发现head元素中有其他元素，但是title元素是最重要的。

如果你在浏览器中看这个文件运行（保存然后刷新），你会发现”my first page”出现在选项卡和窗口的标题栏（不是实际的画布区域）。你在title标签间输入的文本成了文件的标题。如果你把这页加入收藏夹（或者是书签，看浏览器）。你会看见这个标题也出现在这里。
