HTML 初级教程

此章为面向零基础学员。

在学习CSS初级之前，你只要跟着教程循序渐进的学习就行，非常简单。

记住最主要的事情：HTML被用来赋予语意，CSS负责呈现。HTML只是合理架构好内容，视觉呈现由CSS负责。你可能在其他网页上看到不同的学习安排，但是HTML Dog 中文翻译注重的是一开始就进入最佳实践，从一开始就用正确的方式，导致更好的结果的方法。

内容

  • 起步：开始制作第一个页面。
  • 标签，属性和元素：制作HTML页面的基础。
  • 页面标题： 标题。在页面特殊位置显示。
  • 段落： 用段落展示你的内容。
  • 标题： 六个级别的标题。
  • 列表： 定义一个有序和无序列表。
  • 链接：定义一个链接到其他页面。
  • 图片：比文本更多的展现形式。
  • 表格：使用表格组织数据。
  • 表单： 用户提交的内容。
  • 章节回顾：将以上所学内容应用到页面当中。
