起步

大多网页上内容和电脑硬盘里的没两样——只不过一大堆文件按顺序整理成目录。

html文件只不过是一个简单的文本文件，所以开始写HTML.你只需一个简单的文本编辑器。

notepad是一个windows电脑中常见的文本编辑器（常存于 程序>附件 菜单）和mad osx电脑中自带文本编辑TextEdit,或者任何能让你随便尝试文本的程序。

在文本编辑器中键入：


This is my first web page

现在创建一个命名为“HTML”的文件夹存放在电脑任意位置，文件名保存为“myfirstpage.html”

注意!必须要求以扩展名为“.html”保存——有些编辑器，比如notepad，自动默认以“.txt”保存。

还有，你需要确认文件是以纯文本保存。比如文本编辑器总是默认创建一个包含额外格式的多信息文本。在这种情况下，在创建新的文件前，进入文件菜单栏首选项，选择纯文本格式选项。

要查看这些HTML文件，现在还不需要传到服务器上就可以从本地打开。先打开浏览器（Chrome, Firefox, Safari 或者 IE)，在地址栏里输入文件存放在本地的路径比如（c:\html\myfirstpage.html）回车即可。或者从菜单，选择文件打开浏览也行。

这就是第一个网页。开心吗？虽然只是几个刚打的字而已。

我们曾说用最基础的文本编辑器，比如notepad，但你可能听说过ADOBE DREAMWEAVER这样复杂的软件。

当你是新手的时候，要慎用。因为它们可能为了“帮助你”默认使用一些不必要或者不标准的代码。

如果你是认真想学HTML。你应该先阅读如本教程类似的课程，这样对你以后将要学的东西有一个基本的概念。

软件在编辑网页时不会像手写代码那样的随心所欲。

如果你决定用专门的代码编辑软件。我们建议那种还是由你手动编写的软件。它们的文件管理系统和强调代码语法，能让你变得更加专业。
