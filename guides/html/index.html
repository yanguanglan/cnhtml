HTML 教程

HTML 是网页设计最基础的部分， 与CSS不同，HyperText Markup Language 超文本标记语言 可以简单优雅的定义内容结构。

本教程开始使用HTML5。它能展现更多现代浏览器的特性和优点，非常棒。

HTML 初级教程

手把手学习 HTML 基础，新人从这里开始。

  • 起步：开始制作第一个页面。
  • 标签，属性和元素：制作HTML页面的基础。
  • 页面标题： 标题。在页面特殊位置显示。
  • 段落： 用段落展示你的内容。
  • 标题： 六个级别的标题。
  • 列表： 定义一个有序和无序列表。
  • 链接：定义一个链接到其他页面。
  • 图片：比文本更多的展现形式。
  • 表格：使用表格组织数据。
  • 表单： 用户提交的内容。
  • 章节回顾：将以上所学内容应用到页面当中。

HTML 中级教程

一些常见并且很有用的技术技巧。

  • Span和Div：没有特定语意，主要作用于CSS。
  • 文本：缩写， 引用， 编码 格式
  • Meta元标签：页面额外的信息定义。
  • 表格：行和列表：表格基本应用，单元格等。
  • 自定义列表： 自定义一个列表。
  • 文本：地址， 定义， 方向，编辑 格式
  • 区域：语意化定义文档结构。

HTML 高级教程

更标准合理的应用 HTML，比如链接的可访问性等。

  • 文本： 时间， 标记，“展示表现”。
  • 条件注释。
  • 表格： 栏， 抬头，脚注。
  • 可访问性链接。
  • 可访问性表单。
  • HTML5 表单 第一部分：Input 类型。
  • HTML5 表单 第二部分： 属性 和 Date 列表。
  • 可嵌入内容： 视频、语音、画布。
